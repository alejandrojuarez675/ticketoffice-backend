# Especificaciones del Proyecto

Este directorio contiene toda la documentación de especificaciones siguiendo la metodología Specs Driven Development.

## Estructura

- `/features` - Especificaciones de características técnicas
  - `/api/v1` - Endpoints de la API v1
- `/user-stories` - Historias de usuario
- `/acceptance-criteria` - Criterios de aceptación
- `/architecture` - Documentación de arquitectura
- `/decisions` - Decisiones técnicas
- `/style-guide` - Guías de estilo
- `/templates` - Plantillas para documentación

## Cómo contribuir

1. Crear una nueva rama siguiendo Git Flow
2. Documentar la especificación
3. Crear un PR para revisión
4. Una vez aprobado, mergear a develop
